# Design reference

The approved direction is a minimalist European bridal boutique.

## Visual hierarchy

- Serif display headings
- Small uppercase eyebrow text with expanded tracking
- Quiet body typography
- Large portrait dress photography
- Warm neutral canvas
- Dark, nearly black CTAs
- Thin warm-grey separators
- Minimal decoration

## Desktop home layout

1. Announcement bar
2. Header
3. Hero with copy left and portrait image right
4. Three service highlights
5. Featured dress grid, four columns
6. Private fitting callout
7. Newsletter block
8. Dark footer

## Mobile layout

- Compact header with drawer navigation
- Hero content and image stack cleanly
- Featured cards use one or two columns depending on width
- Buttons remain comfortably tappable
- No horizontal overflow

## Tone of copy

Refined and direct. Avoid clichés, excessive romance, exclamation marks and generic luxury jargon.
